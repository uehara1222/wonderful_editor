# WonderfulEditor

## 概要

Qiita風の記事作成アプリ

学習した内容のアウトプットの為に作成しました。

簡単に記事の作成ができ、ユーザー同士で共有することができます。


## 動作環境

* Ruby: 2.7.2
* Rails: 6.0.6.1
* Node: 15.7.0
* Vue.js: 2.6.11
* DB: PostgreSQL

## 機能一覧

* ユーザー登録
* ログイン、ログアウト
* マイページ機能
* 記事の下書き機能
